# ÚRK – posudok vedúceho BP – LaTeX šablóna

LaTeX šablóna pre posudok vedúceho bakalárskej práce platná na Ústave robotiky a kybernetiky FEI STU.

## Pokyny pre vyplnenie
* [`main.tex`](main.tex) – Nastavenie základných údajov
    -  Meno študenta
    -  Študijný odbor (predvolené)
    -  Názov bakalárskej práce
    -  Meno vedúceho BP
    -  Index CRZP
    -  Dátum
* [`hodnotenie_body.tex`](include/hodnotenie_body.tex) – Vyplnenie bodového hodnotenia pre jednotlivé kategórie. Suma bodov sa vyhodnocuje automaticky.
* [`hodnotenie_slovne.tex`](include/hodnotenie_slovne.tex) – Vyplnenie slovného hodnotenia (súvislý text).
* [`stanovisko_originalita.tex`](include/stanovisko_originalita.tex) – Zhodnotenie výsledku kontroly originality (prvá veta je vypísaná automaticky).

Známka a odporúčanie na obhajobu sú doplnené automaticky.
